# Financial System Entities

Financial System Entities is a Java-based project that defines the core entities required for a financial management system.  
This project demonstrates the implementation of Object-Oriented Programming (OOP) principles to model users, accounts, and transactions in a structured and scalable way.

---

## Features

- User Entity implementation  
- Account Entity implementation  
- Transaction Entity implementation  
- Proper relationship mapping between entities  
- Clean and structured Java code  
- Beginner-friendly project structure  

---

## Technologies Used

- Java  
- Object-Oriented Programming (OOP)  
- IntelliJ IDEA / Eclipse / VS Code
